# Customer Review Sentiment Classifier

## Project Overview
Built a sentiment classification system using 
real Amazon India product reviews.

## What I Did
- Collected 50 real Amazon customer reviews
- Manually annotated reviews into 4 categories:
  POSITIVE, NEGATIVE, NEUTRAL, UNCLEAR
- Trained a Naive Bayes classifier using 
  TF-IDF vectorization
- Achieved 90% model accuracy

## Tools Used
- Python
- Pandas
- Scikit-learn
- Microsoft Excel

## Results
- Total Reviews: 50
- Model Accuracy: 90%
- Issues Flagged: 1
